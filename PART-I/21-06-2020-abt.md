# Abstraktit sidontapuut

Abstraktit sidontapuut lisäävät [abstrakteihin syntaksipuihin](21-06-2020-ast.md) nähden
mahdollisuuden uusien muuttujien ja parametrien lisäämiseen.

Jokaiseen uuteen muuttujaan tai parametriin liitetään niiden vaikutusalue (scope), jolla niihin voidaan viitata.
Tyypillinen esimerkki muuttujan vaikutusalueesta imperatiivisessa kielessä olisi esimerkiksi:

```c
int a = 1;
{
  int a = 2;
  a = 3;
}
```

Tässä esimerkissä luodaan kaksi erillistä 'a' nimistä muuttujaa, joista jälkimmäisen arvo muutetaan.
Kun poistutaan aaltosulkujen alueelta, lakkaa myös toinen 'a' olemasta ja viittaukset tarkoittavat ensimmäisenä alustettua.
Näin ollen muuttujien lukumäärä voi abstraktissa sidontapuussa olla suurempi syvemmällä rakenteessa kuin sen ulkopuolella.

Harper nostaa esille olennaisimpana ideana sen, että tunnisteet (identifiers) tulisi ymmärtää viittauksina niiden sidontaan.
Näin ollen itse tunnisteiden nimet ovat merkityksettömiä, kunhan jokainen tunniste voidaan liittää yksikäsitteisesti johonkin sidontaan.

Tiedostossa [koodit/ABT.hs](koodit/ABT.hs) on yksinkertainen esimerkkitoteutus.
